# Expression Calculator-v1.0
## 项目简介
这是一个使用 **C++ 编写的命令行表达式计算器**。
用户可以在终端中输入数学表达式，程序会自动解析并计算结果。
该程序支持常见的数学运算符、括号以及小数运算，并且可以处理简单的一元运算（例如 -5）。
程序会持续运行，直到用户输入 exit 退出。
## 功能特点
本计算器支持以下功能：
* 基本运算
  * 加法 +
  * 减法 -
  * 乘法 *
  * 除法 /
* 进阶运算
  * 取模 %
  * 幂运算 ^ 或 **
* 支持括号
  例如：
  (3 + 5) * 2
* 支持小数
  3.5 * 2
* 支持一元运算
  -5 + 3
  +7
* 错误检测
  * 除以 0
  * 表达式格式错误
  * 括号不匹配
## 示例
程序运行示例：
Expression Calculator (type 'exit' to quit)
> 3+5*2
13
> (3+5)*2
16
> 2^3
8
> 10%3
1
> exit

## 项目说明
该项目主要实现了一个 **表达式解析与计算系统**。
程序首先将输入的表达式转换为 Token，然后使用栈算法将 **中缀表达式转换为后缀表达式**，最后再计算后缀表达式得到结果。
该算法是一种经典的表达式计算方法。
